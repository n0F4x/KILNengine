# Coding Guidelines

## ClangFormat

Please format your code using the provided _.clang-format_.
We are all free to propose changes. 

## ClangTidy

Please follow the practices setup in the provided _.clang-tidy_.
We are all free to propose changes.
Example projects can deviate from these rules.

## Sanitizers

It is good practice to use compiler sanitizers for development.
Check out [this](https://docs.conan.io/2/security/sanitizers.html) page for more information about how to enable them using Conan.
